# ROADMAP - Projeto BI Portfolio
# Tema: Análise de E-commerce - Marketplace Performance & Churn

Objetivo:
Construir um pipeline completo de dados usando Python, Google BigQuery e visualizações em Looker Studio e Power BI.
Resolver um problema real de negócio:
  - Reduzir churn
  - Melhorar performance dos vendedores
  - Otimizar logística e experiência do cliente

Dataset:
- Brazilian E-Commerce Public Dataset by Olist (Kaggle: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce?resource=download)
